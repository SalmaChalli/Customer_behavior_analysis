# Customer_behavior_analysis
This project represents a complete analysis of customer's behavior. It encompasses all the steps of data analysis from data preparation and modeling to insight generation and visualization.
# Project overview
- Data Preparation,Modeling & Exploratory Data Analysis (Python): Clean and transform the raw dataset for analysis.
- Data Analysis (SQL): Simulate business transactions, and run queries to extract insights on customer segments, loyalty, and purchase drivers.
- Visualization & Insights (Power BI): Build an interactive dashboard that highlights key patterns and trends, enabling stakeholders to make data-driven decisions.
# Project files
- Business problem.pdf :Defines the business problem and project objectives
- customer_shopping_behavior.csv: raw dataset.
- Customer_shopping_behavior.ipynb: Data cleaning, transformation, and EDA.
- Business_analysis.sql: SQL queries for business insights.
- customer_behavior_dashboard.pbix: Interactive Power BI dashboard
